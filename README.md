
Uses the [JCEF Maven libraries](https://github.com/jcefmaven/jcefmaven) in a [Spring Boot](https://spring.io/projects/spring-boot) application.

## Build and run

```shell
./gradlew bootRun
```

## Build runnable jar

```shell
./gradlew bootJar
```

where the built jar is located in `build/libs`.
